Atalhos VS CODE

CTRL + O: para abrir um arquivo;

CTRL + K + CTRL + O: para abrir uma pasta;

ctrl + n: para criar um novo arquivo + ctrl + s para salvar o arquivo na pasta;

CTRL + win + alt + n: o VS CODE irá abrir uma caixinha de texto no topo para que seja digitado o nome do novo arquivo;

ctrl + bpara fechar/abrir a aba do explorer do VS Code.

ctrl + wpara fechar a aba do arquivo que está seleconado;

ctrl + k + wpara fechar todas as abas de arquivos abertas;

alt + nºpara navegar para a aba referente ao nº que você digitou;

ctrl + ppara abrir a caixa de pesquisa do VS Code e localizar algum arquivo;

!para criar uma estrutura HTML básica;

elemento.classepara criar um elemento com uma class: classe;

elemento#idpara criar um elemento com um id: id;

elemento*nºcriar um nº de elementos referente ao nº que você digitou;

elementopai>elementofilhopara criar um elemento, já com outro elemento dentro;

Juntando tudo, podemos chegar no seguinte atalho:

elementopai.classe#id*2>elementofilho.classe#id*2

cria um nº de elementos pai de class: classe e id: id, cada um com um nº de elementos filhos de class: classe e id: id.

Essa multiplicação de elementos também serve para qualquer outro tipo de elemento ou texto.

p>loremcria um parágrafo com um texto lorem dentro;

ctrl + shift + kdeleta toda a linha em que o cursor extiver localizado.

ctrl + shift + fabrir a janela de busca global (em todo o projeto).

ctrl + fabrir a janela de busca na aba do arquivo onde você se encontra.

Para substituir um texto, elemento, atributo, valor etc. Basta fazer o seguinte:

ctrl + fabrir a janela de busca na aba do arquivo onde você se encontra.

>para abrir a opção de substituir o texto.

ctrl + alt + Enterpara confirmar a substituição de todos os itens.

Há uma outra maneira de substituir mais de um item de uma só vez:

alt + clickpara ir colocando a seleção do cursor em todas as áreas do código que forem ser substituídas.

Após isso, é só apagar e ir reescrevendo em um ponto que a mudança irá sendo realizada em todos os outros pontos selecionados.

crtl + shift + ↑seleciona linhas para cima.

crtl + shift + ↓seleciona linhas para baixo.

alt + ↑deslocar uma linha para cima.

OBS: se várias linhas estiverem selecionadas este atalho deslocará todas elas.

alt + shift + ↑copia uma linha para cima.

OBS: se várias linhas estiverem selecionadas este atalho copiará todas elas.

alt + ↓deslocar uma linha para baixo.

OBS: se várias linhas estiverem selecionadas este atalho deslocará todas elas.

alt + shift + ↓copia uma linha para baixo.

OBS: se várias linhas estiverem selecionadas este atalho copiará todas elas.

ctrl + ]divide a tela em abas diferentes.

ctrl + nºpara alternar o foco da aba em tela dividida.

Exmplo:

ctrl + 1 para a primeira tela, ctrl + 2 para a segunda e assim sucessivamente.

ctrl + glocaliza uma linha. Uma caixa de texto abrirá no topo para que seja especificado qual o nº da linha desejada.

ctrl + shift + olocalizar uma função ou variável.
